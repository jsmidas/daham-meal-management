GCP 서버에서 다음 명령어를 실행하여 최신 버전으로 업데이트:

```bash
# 1. 기존 컨테이너 중지
sudo docker-compose down

# 2. 최신 코드 가져오기
git pull origin master

# 3. Docker 이미지 재빌드 및 실행
sudo docker-compose up -d --build

# 4. 서비스 상태 확인
sudo docker-compose ps
sudo docker-compose logs
```
