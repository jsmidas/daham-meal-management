  - 최소 사양: CPU 2코어, RAM 4GB, 디스크 20GB
  - 비용: 월 2-10만원
- [ ] Ubuntu 서버 설치
- [ ] 방화벽 설정 (포트 22, 80, 443만 열기)

```bash

sudo apt install python3 python3-pip nginx sqlite3 git -y
sudo apt install certbot python3-certbot-nginx -y
```
